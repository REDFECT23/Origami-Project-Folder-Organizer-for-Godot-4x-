# Origami: Project Folder Organizer for Godot 4.x

A powerful productivity tool that keeps your project hierarchy clean and professional.

## Features
- **Auto-Sort:** Move files to designated folders based on extensions.
- **Deep Clean:** Recursively removes empty folders left after moving files.
- **Project Templates:** Create a standard folder structure (scenes, scripts, assets) with one click.
- **Customizable:** Add your own rules and ignore specific folders via a built-in UI.
- **Save/Load:** Your settings are stored within the project.

## Installation
1. Copy the `addons/project_organizer` folder to your project's `addons` directory.
2. Go to **Project Settings** > **Plugins** and enable **Origami: Project Folder Organizer**.
3. Look for the "Project Organizer" tab in the bottom-right dock.
